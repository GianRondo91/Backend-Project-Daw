## 📄 Sobre el proyecto <a name = "about"></a>
Este proyecto fue creado para facilitar la reserva de instalaciones.

El usuario podra:
- Registrarse
- iniciar Sesión
- Modificar sus datos
- Realizar una reserva
- Visualizar sus reservas
- Eliminar sus reservas
